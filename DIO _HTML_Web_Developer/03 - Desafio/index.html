Aula 01 - Estrutura Básica do HTML

[inicio]

Nesta primeira aula entendemos como o HTML está estruturado. Um página HTML, em sua forma mais básica, é a combinação de tags e atributos dessas tags.

Aula 02 - Falando sobre Tags

[inicio]

O esqueleto do HTML está basicamente organizado por meio das seguintes tags:

  • <html> </html>, que delimitam todo o código da página;
  • <head> </head>, que carrega informações do cabeçalho da página. Ex: o título desta página Meu Estudo HTML
  • <body> </body>, dentro dessas tags estarão todo o texto que o usuário irá ver.
Outras tags que geralmente aparecem no texto são:
  • <img >, responsável por inserir uma imagem no texto.
  • <input >, que abre uma caixa de entrada de texto (pode ser número também.
  • <strong >, que deixa o texto em negrito (a tag <b> </b> também realiza o mesmo procedimento).
  • <i> </i>, deixa o texto em itálico, tal como esse texto aqui.
  • <h1> </h1>, refere-se ao títulos (cabeçarios), presentes na página. Essa tag vai de h1 até h6. .
  • <hr>, serve para inserir uma linha no texto.

Aula 03 - Atributos das Tags

[inicio]

Na terceira aula tratatmos de alguns atributos das tags vistas anteriormente. Os atributos são palavras especiais que são usados para controlar o comportamento do elemento. A estrutura de um atributo é nome="valor", onde nome é um atributo e valor é o valor do atributo.

Os atributos se apresentam com as tags da seguinte maneira:

  • <img width="500" src"endereco_imagem" alt="nome_imagem">, esta estrutura está presente nessa página.
  • <h2 id="aula-01">Aula 01 - Estrutura Básica do HTML" </h2>, esta estrutura está presente nessa página.

Aula 04 - Textos

[inicio]

Nesta aula aprendemos um pouco mais sobre as atags e atributos que podem ser usados no texto. Outras tags vistas aqui foram:

  • <blockquote> </blockquote>, que mostra o texto como uma citação, conforme abaixo;

  • <mark> </mark>, que faz uma marcação no texto;
  • <sup> </sup>, que faz uma elevação no texto;

Aula 05 - Lista ordenadas e não ordenadas

[inicio]

Nesta última aula aprendemos sobre as listas ordenadas e não ordenada. Por meio delas podemos inserir no texto itens de listas. Eles podem ser numerados ou apenas conter marcadores. As tags que representam as listas são:

Lista não-ordenada.:
  • <ul> </ul>, significa unordered list ou lista não-ordenada;
  • <ol> </ol>, significa ordered list ou lista ordenada;
  • <li> </li>, significa item in a list ou item em uma lista. É preciso sempre que tag ul ou ol for chamada;
  • Esses pontos são lista não-ordenadas
